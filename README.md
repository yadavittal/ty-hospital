# TY Hospitals #

This is an application for Hospitals to manage their business staff online.

### What is this repository for? ###

* TY Hospitals
* Version: 1.0

## Getting Started

### Tech Stack ###
* Vite + React + Typescript
* React Router v7.0+
* Sass/scss
* Tailwindcss 4.0+
* Redux Toolkit
* Axios

### Prerequisites

- Node.js 21+
- npm 9+

### Installation

```bash
# Install dependencies
npm install

# Start development servers
npm run dev

#Build for development
npm run build:dev

# Build for production
npm run build:prod

# Install Serve globally
npm i --global serve

# Serve production build
npm run serve:prod
```
